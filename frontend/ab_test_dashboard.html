Methodology

How this evaluation works

Dataset
FiQA (Financial Opinion QA) via BeIR — 57k financial forum passages, 648 labelled test queries. 100 queries and 5k corpus chunks used in this run.
Systems
System A: BM25-only (rank-bm25, same tokeniser). System B: BM25 + sentence-transformers (all-MiniLM-L6-v2) fused via RRF (k=60).
Metrics
Precision@K, Recall@K, MRR, NDCG@K for retrieval quality. Latency is wall-clock search time per query.
Statistics
Paired Wilcoxon signed-rank test — non-parametric, robust for small/skewed samples. Effect size via Cohen's d. 95% CIs via bootstrap (2000 iters).
RRF Formula
RRF(d) = Σ 1/(k + rank_i) with k=60. A chunk ranked #1 in both lists scores ≈ 0.033 — the theoretical maximum.
Verdict
All 4 metrics significant (p ≤ 0.0013). MRR and NDCG show medium effect sizes (Cohen d ≥ 0.55). Hybrid is the clear winner on FiQA.
